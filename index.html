What is the world wide web?

A bunch of computers that communicate with each other by sending an HTTP request to a server. When a person goes to a web page like www.google.com, the server finds said HTML document and sends it back to the user's computer, web browsers like safari, google chrome, or mozila foxfire display the html document that we recognize as web pages.


Well, what's HTML?

HTML stands for HyperText Markup Language. Almost all documents we see on the web are HTML. These documents contain text that describes "what you see" and markup that describe "how it looks".


Tags and Elements

HTML documents are made of elements that tell the browsers that tell browsers what they are by using tags.


Inline vs Block Elements

HTML elements are either inline or block. Block elements form an "invisible box" around the content inside of them while inline elements simply apply to the code they surround without forming a new box around the content.
